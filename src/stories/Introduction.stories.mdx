# Welcome to SKA-GUI-COMPONENTS within Storybook

Storybook helps you build UI components in isolation from your app's business logic, data, and context.
That makes it easy to develop hard-to-reach states.
Save these UI states as **stories** to revisit during development, testing, or QA.

This library provides direct usage of the components and styling available from <strong>ska-javascript-components</strong>

- <strong>Theme</strong>....... Used directly to provide a standard theme for use by Material-UI. ( <strong>
    THEME_DARK
  </strong> & <strong>THEME_LIGHT</strong> constants available )
- <strong>Colors</strong>....... Theme colors, should use of the <strong>Theme</strong> component
  not be viable
- <strong>Styling</strong>....... Theme styling, should use of the <strong>Theme</strong> component
  not be viable
- <strong>Logo</strong>.......... Standard SKAO Logo, supporting light & dark modes. ( <strong>
    LOGO_DEFAULT_HEIGHT
  </strong>) available )
- <strong>Spacer</strong>...... Component so that spacing can be set if needed ( <strong>
    SPACER_HORIZONTAL
  </strong> & <strong>SPACER_VERTICAL</strong> available )
- <strong>Status</strong>....... Component providing a Engineering shape that is fully accessible
  depending upon property provided

Below is a short summary of the components contained directly via this library, with examples provided via the left menu
